CASE STUDY

Docker is an open-source project that automates the deployment of applications inside software containers, by providing and additional layer of abstraction and automation of operating system-level virtualization on Linux.

Mesosphere combines your datacenter servers and cloud instances into one shared pool from which any app or framework can draw.

Apache Mesos is a cluster manager that simplifies the complexity of running applications on shared pool of servers.
